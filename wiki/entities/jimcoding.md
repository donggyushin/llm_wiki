# 짐코딩

## 핵심 요약

한국어 YouTube 개발 교육 채널 운영자 + 인프런 **"클로드 코드 완벽 마스터"** 강의 저자. Claude Code 의 **메모리 파일 · 컨텍스트 엔지니어링 실전 팁** 을 공식 문서 + 오픈소스 분석 기반으로 풀어 설명하는 것이 특징. 인프런 강의는 출시 4개월 만에 4,300+ 수강 (개발자 · 마케터 · 데이터 분석가 · 비개발자 혼재).

## 콘텐츠 특징

- **공식 문서 + 오픈소스 뜯기** — 추상적 이론 대신 Trigger.dev / CockroachDB 같은 인기 오픈소스에서 실제 `.claude/rules/` 사용 사례를 직접 추출해 보여줌
- **비유 중심 설명** — "라떼 레시피", "맥도날드 직원별 매뉴얼" 등 실무 맥락을 일상 비유로 풀어냄
- **실전 팁** — 본인 프로젝트 + 오픈소스 분석에서 체득한 내용 강조 ("실제로 사용하는 방법")
- **시리즈 구성** — 한 주제를 2~3 부작으로 분할 ("대규모 컨텍스트 엔지니어링" 3부작 등)
