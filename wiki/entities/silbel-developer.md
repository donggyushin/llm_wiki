# 실벨 개발자

## 핵심 요약

한국어 YouTube 개발 교육 채널 운영자. **Claude Code · 에이전틱 엔지니어링 · 하네스 엔지니어링** 을 주제로 체계적 강의를 제공. 단순 기능 소개가 아니라 **AI 개발 방법론의 4축 프레임워크**를 종합 정리하는 메타 관점의 콘텐츠가 특징. 영상마다 Notion 치트시트를 병행 배포.

## 콘텐츠 특징

- 한국어 강의, 기술 용어는 원어 유지
- **메타포 기반 설명** (말과 마구, 세컨드 브레인, 총알로서의 토큰)
- 실전 데모 위주 — 단독 기능 설명보다 **통합 워크플로우** 에 집중
- 주요 레퍼런스: Anthropic 공식 문서, OpenAI 블로그, Martin Fowler

## 반복 강조 원칙

두 영상 공통으로 반복되는 원칙:

1. **컨텍스트 위생이 품질을 결정** — 200k 토큰은 금방 찬다
2. **플랜 먼저, 구현 나중** — 플랜 세션과 구현 세션 분리
3. **Opus 는 메인, Sonnet/Haiku 는 서브** — 토큰이 총알
4. **AI 추론 ↔ 코드 실행 분리** — WAT 프레임워크, 하네스의 Execution Loop
5. **프롬프트를 고치지 말고 시스템을 고쳐라** — 하네스 철학의 핵심
