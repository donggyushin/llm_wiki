# Matt Pocock

## 핵심 요약

영국 출신 TypeScript 교육자이자 AI 코딩 코스 운영자. **Total TypeScript** · **AI Hero** (aihero.dev) 운영. AI Engineer 컨퍼런스(2026) 발표 *"Software Fundamentals Matter More Than Ever"* 에서 **AI 시대일수록 펀더멘털이 더 중요하다** 는 테제를 펼쳤다 — [[sources/matt-pocock-software-fundamentals]]. *Claude Code for Real Engineers* 강좌 + GitHub `mattpocock/skills` 리포지토리에서 AI 코딩용 스킬을 공개한다.

## 주요 활동

- **TypeScript 교육**: Total TypeScript 강좌 시리즈로 잘 알려져 있음.
- **Claude Code for Real Engineers**: AI 코딩 강좌. 도발적인 강좌명 의도적으로 채택.
- **AI Hero (aihero.dev)**: 뉴스레터·강좌·자료 허브.
- **mattpocock/skills**: AI Engineer 발표에서 소개한 스킬을 모은 GitHub 리포지토리. *grill me*, *ubiquitous language*, *improve codebase architecture* 등 수록.
