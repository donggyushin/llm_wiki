# 사물 존칭·과한 격식

## 핵심 요약

예의처럼 보이지만 실제로는 어색하고 결례인 표현 묶음. "커피 나오셨습니다 / 말씀이 계시겠습니다 / 외람되지만 / 송구하오나" 등.

두 갈래로 나뉜다:
1. **사물 존칭** — 사람이 아닌 것에 존칭 어미를 붙이는 비문 ("커피"가 어떻게 "나오셔" 하나)
2. **과한 격식** — 사극 톤·시대극 톤으로 일상 업무를 무겁게 만드는 표현 ("외람되지만/송구하오나")

처방: **존칭은 사람에게만**. 격식은 *사극 톤* 이 아니라 *정확하고 단정한 문장* 으로 표현.

---

## 5개 대표 항목 ([[sources/business-writing-30-mistakes]] 항목 21~25)

### 21. 커피 나오셨습니다 → 커피 나왔습니다
사물 존칭의 대표 사례. "커피"는 사람이 아니므로 "-시-" 가 붙을 수 없다.
같은 오류 패턴:
- 보고서가 완성되셨습니다 → 완성되었습니다
- 회의실이 비어계십니다 → 비어 있습니다
- 자료가 도착하셨습니다 → 도착했습니다

> 서비스업 과잉 존칭이 사무 메일까지 옮겨간 사례. 받는 쪽은 *어색함* 만 느낀다.

### 22. 말씀이 계시겠습니다 → 말씀이 있겠습니다
"계시다" 는 사람 존칭. *말씀* 은 사물(추상명사). 따라서 "말씀이 있다" 가 맞음.
주체 존대를 높이고 싶다면: "팀장님께서 말씀하시겠습니다".

### 23. 수고하셨습니다 (윗사람에게) → 결례
원칙적으로 *수고* 는 윗사람 → 아랫사람 방향의 노고 인정 표현. 윗사람에게 쓰면 결례.

대안:
- 고생 많으셨습니다
- 감사합니다
- 잘 부탁드립니다 (다음 단계로 넘기는 맥락)

> [!note] 실제 사용 현실
> 한국 직장에서 *수고하셨습니다* 가 윗사람에게 광범위하게 사용되고 있어 결례로 받아들여지지 않는 경우도 많다.
> 그러나 *공식 메일·정중한 자리에서는 피하는 것이 안전*.

### 24. 외람되지만 → 한 가지 의견 드리면 / 한 가지 여쭤보겠습니다
"외람" 은 *분수에 넘치게 함부로* 를 뜻하는 무거운 한자어. 제안 하나 하는데 사형 선고처럼 됨.
대안:
- "한 가지 의견 드리면"
- "한 가지 여쭤보겠습니다"
- "혹시 다른 관점도 있을까 싶어 공유드립니다"

### 25. 송구하오나 → 양해 부탁드립니다 / 죄송합니다만
같은 이유로 과함. 사극 톤. 가벼운 일을 무겁게 만든다.
"송구하오나 일정 변경이 가능할까요" → "죄송합니다만 일정 변경 가능할까요" 또는 "양해 부탁드립니다, 일정 변경이 가능할지요"

## 두 안티패턴 진단표

### 사물 존칭 진단

다음 패턴 발견 시 의심:

| 패턴 | 교정 |
|------|------|
| `(사물) + 나오셨/계시/되셨` | -시- 제거 |
| `~이/가 있으십니다` (사물 주어) | `~이/가 있습니다` |

> [!warning] 헷갈리기 쉬운 경계
> "주문하신 커피 나왔습니다" 는 OK (주체는 *주문하신 분* — 사람).
> "커피 나오셨습니다" 는 X (주체는 커피 — 사물).

### 과한 격식 진단

다음 표현 발견 시 검토:

| 표현 | 무게 | 권장 |
|------|------|------|
| 외람되지만 | 너무 무거움 | "한 가지 의견 드리면" |
| 송구하오나 | 사극 톤 | "죄송합니다만" |
| 황공하오나 | 사극 톤 | (거의 사용 X — 영화 대사) |
| 감히 ~ | 무거움 | (대부분 삭제) |
| 분에 넘치는 | 너무 겸손 | (대부분 불필요) |

## 격식의 진짜 출처

격식 = 사극 톤이 아니다. 격식의 진짜 출처는:

1. **정확한 호칭** — "팀장님" 한 번이 "외람되지만" 다섯 번보다 정중함.
2. **단정한 문장 구조** — 한 문장에 한 메시지. 늘어지지 않게.
3. **구체적 마감/관점/대상** — [[concepts/action-spec-three-elements]].
4. **마무리 인사** — 단정한 "감사합니다" 한 줄.

→ [[concepts/business-writing-clarity]] 의 *명확성=친절* 따름정리 2 *"격식은 무게가 아니라 정확성"* 직접 사례.
