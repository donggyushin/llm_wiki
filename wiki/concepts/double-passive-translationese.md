# 이중 수동·번역체

## 핵심 요약

한국어 직장 글쓰기의 1순위 안티패턴. 두 갈래로 나뉜다.

1. **이중 수동** — "되다"·"보이다" 처럼 이미 수동 의미를 가진 동사에 다시 "어집니다"가 붙는 비문
2. **번역체** — 일본어 「において」, 영어 "have a meeting" 같은 외국어 통사를 직역해 옮긴 표현

둘 다 **격식 차린 척** 효과만 주고, 의미는 동일하거나 오히려 흐리멍덩해진다.
삭제·치환만 해도 글이 깔끔해지고 책임 주체가 명확해진다.

---

## 5개 대표 항목 ([[sources/business-writing-30-mistakes]] 항목 1~5)

### 1. 되어집니다 → 됩니다
"되다"가 이미 수동. "어집니다"가 또 붙으면 비문. 같은 원리:
- 만들어집니다 → 만듭니다 / 만들어집니다 (목적어 따라 다름)
- 쓰여집니다 → 쓰입니다

### 2. 보여집니다 → 보입니다
"보이다"에 수동을 한 번 더 붙인 형태. *보다*의 피동이 *보이다* 이므로 이중 피동.

### 3. 사료됩니다 → 생각됩니다 / 판단됩니다
"사료(思料)"는 일제 잔재 한자어. 격식 효과만 있고 의미는 동일.
- 변형: "여겨집니다" → "여깁니다" / "생각합니다"

### 4. ~에 있어서 → ~에서 / ~에 / ~의 경우
일본어 「において」 직역. "업무에 있어서" → "업무에서". 빼면 99% 더 깔끔.
- "프로젝트에 있어서 가장 중요한 것은" → "프로젝트에서 가장 중요한 것은"

### 5. 회의를 가졌습니다 → 회의했습니다
영어 "have a meeting" 직역. 같은 함정:
- "시간을 가지다" → "시간을 보내다" / 그냥 "기다리다"
- "대화를 가지다" → "대화하다"

## 진단 패턴

다음 어미·표현 발견 시 의심:

| 패턴 | 교정 방향 |
|------|----------|
| `~어집니다` (이미 피동인 동사 뒤) | 능동 어미로 교체 |
| `~에 있어서` | `~에서` / `~의 경우` / 삭제 |
| `~을(를) 가지다` (have ~ 직역) | 동작 동사로 직접 표현 |
| 한자어 "사료/감안/도모/제고" | 고유어로 교체 |

---

## 왜 1순위 안티패턴인가

세 가지 동시 효과:

1. **책임 회피** — 수동태는 행위자를 가린다. "보고서가 작성되어집니다" 는 누가 쓰는지 모호.
2. **의미 중복** — 이중 수동은 같은 의미를 두 번 입힌다. 정보량은 그대로, 글자만 늘어난다.
3. **격식 위장** — "사료됩니다" 는 격식 차림처럼 보이지만 실제로는 일제 잔재 한자어.

→ [[concepts/business-writing-clarity]] 의 *명확성=친절* 원칙 정면 위반.
